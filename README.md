# WifiCracker
This tool gives you a wifi password by cracking the password

# How To Installition
* ` git clone https://github.com/MhnXp/WifiCracker.git `
* `pip install comtypes`
* `pip install pywifi`
* `pip install -r requirments.txt`
* `cd WiFi Cracker `
* `python wifi-cracker.py `
